# Prompt Log

This directory supports the KSE AI-assisted development log. It contains short summaries of the main development stages and placeholders where real Codex prompts, outputs, screenshots, or repair notes can be pasted later.

This is not a fake full chat export. Keep entries concise, source-aware, and clear about what was requested, what was implemented, what failed, and how it was repaired.

Suggested use:

- Add the exact user prompt or a screenshot when needed.
- Add command outputs only when they matter for evidence.
- Do not paste API tokens or private secrets.
- Keep anomaly analysis language exploratory.
